Clarity OS T4 - Quick Start Guide (2 minutes)

=== What This Does ===
Search your documents like Google, but 100% private and local.
- Find exact text in any document
- Get file locations and snippets
- Search Word docs, PDFs, text files, code files
- Never changes your files (read-only)

=== How to Use ===

1. Double-click "Run Search.bat"
2. Follow the friendly prompts
3. Results are saved automatically

That's it! No technical knowledge needed.

=== What Happens ===

1. It asks for the folder to search
   - Remembers your choice for next time
   - Use "." for current folder
   - Use "Documents" for your Documents folder

2. It asks what you're looking for
   - Choose "Search" for exact text
   - Choose "Ask" for questions with summaries

3. It searches and saves results
   - Answer saved to: results\last_answer.txt
   - Sources saved to: results\last_sources.txt

=== Example Searches ===

Find company policies:
  Search for: "privacy policy"

Find contract terms:
  Search for: "termination clause"

Ask questions:
  Ask: "What are our security policies?"

=== Results ===

After searching, you can:
- View the answer immediately
- See all sources and citations
- Search again
- Exit and check results later

=== Tips ===
- First run takes a moment to set up
- Subsequent searches are faster
- Your folder choice is remembered
- All results are saved in the results folder

=== Privacy ===
100% Local. Nothing leaves your computer.
No internet connection needed.
Your data stays yours.

=== Need Help? ===
Just follow the prompts - they guide you through everything!
The system is designed to be user-friendly and intuitive.

=== Done! ===
That's everything you need to search your documents privately and efficiently.
